# Toronto SOC Automation Toolkit

**Portfolio Project** — April 2026

---

### 🚀 Purpose

Demonstrates real-world SOC Analyst and Threat Hunter skills targeted at the **Toronto cybersecurity job market**, including:

- Structured log generation (Splunk-like)
- Regex-based threat detection (SIEM/ELK style)
- Real-time log monitoring
- Port scanning
- Enterprise password strength validation

---

### ✨ Features

- **Log Writer**: Creates structured security events with validated severity levels
- **Threat Detector**: Scans logs using regex patterns for brute force, suspicious geo, phishing, privilege escalation, and data exfiltration
- **Real-Time Monitor**: Live tailing with inline threat alerting (`tail -f` style)
- **Port Scanner**: Basic network reconnaissance tool
- **Password Checker**: Compliance-focused strength evaluator (PIPEDA / financial sector standards)

- File I/O + real-time monitoring
